文字垂直居中

在 <Text></Text> 标签外面套一层 <View> 标签，在 <View> 中设样式 alignItems: 'center' 、 justifyContent: 'center' 即可。

默认颜色

有时候真的需要知道 react-native 的那个默认背景色是什么，对，就是那个灰不拉几的颜色，可以直接用QQ的截图工具，鼠标移过去即可。

颜色为 RGB: (233,233,239) 或 RGB: #E9E9EF
