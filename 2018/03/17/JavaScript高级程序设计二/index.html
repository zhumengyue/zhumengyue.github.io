Array 相关方法

栈 / 队列方法

提供四种方法：

push() : 接收任意数量参数，逐一添加到数组末尾，返回修改后数组长度。

pop() : 从数组末尾移除一项，减少数组 length 值，返回移除的项。

shift() : 移除数组中第一个项，并返回该项。

unshitf() : 在数组开头添加任意个项，返回修改后数组长度。

重排序方法

reverse() : 反转数组项顺序

sort() : 默认情况下按升序（最小值位于前面），会调用每个数组项的 toString() 方法

显然，这种 sort() 方案在多数情况下都不是最佳方案，因此 sort() 方法可以接收一个比较函数作为参数，以便指定哪个值位于哪个值前面

操作方法

concat() : 先创建当前数组的一个副本，将接收到的参数添加到副本末尾

slice() : 基于当前数组的一个或多个项创建一个新数组，接收1~2个参数，即要返回项的起始和结束位置。不会影响原数组。

注意：如果只有一个参数，则返回从指定位置开始到当前数组末尾的所有项。如果有两个参数，返回起始位置和结束位置之间的项——但不包含结束位置的项。

  • 如果起始位置大于结束位置，返回空。

  • 如果参数中有负数，则用数组长度加上该数来确定位置

    如一个包含5项的数组，调用 slice(-2,-1) 与调用 slice(3,4) 得到的结果相同

splice() :

  • 删除：可以删除任意数量的项，需要两个参数：要删除的第一项的位置和要删除的项数。splice(0,2) 会删除前两项。
  • 插入：向指定位置插入任意数量的项，需要三个参数：起始位置、0（要删除的项数）、要插入的项
  • 替换：向指定位置插入任意数量的项，同时删除任意数量的项，需要三个参数：起始位置、需要删除的项数、要插入的项

splice() 始终返回一个数组，包含从原数组中删除的项。

Date 相关方法

基本方法

Date()

在不传入参数的情况下，默认返回当前日期和时间。

如果传入参数，必须传入表示该日期的毫秒数。为了简化这一过程，ECMAScript 提供两个方法：Date.parse() 、Date.UTC() 。

Date.parse()

接收一个表示日期的字符串参数，然后尝试根据这个字符串返回相应日期的毫秒数。
