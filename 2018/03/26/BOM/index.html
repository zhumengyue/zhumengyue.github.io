window 对象

Firefox使用 screenX 和 screenY 提供窗口位置信息

IE、Safari、Opera、Chrome使用 screenLeft 和 screenTop 提供窗口位置信息

innerHeight 、innerWidth 返回页面视图区大小

outerHeight 、outerWidth 返回浏览器窗口本身尺寸

IE、Firefox、Safari、Opera、Chrome中， document.documentElement.clientWidth 和 document.documentElement.clientHeight 保存页面视口信息。

location 对象

location 对象既是 window 对象的属性，也是 document 对象的属性，即 window.location 和 document.location 引用的是同一对象。

属性名 例子 说明
hash “#contents” 返回URL中的hash（#后跟零或多个字符），如果URL中不包含散列，则返回空串
host “www.wrox.com:80” 返回服务器名称和端口号（如果有）
hostname “www.wrox.com” 返回不带端口号的服务器名称
href “http://www.wrox.com“ 返回当前加载页面的完整URL
pathname “/WileyCDA” 返回URL中的目录和（或）文件名
port “8080” 返回URL中指定的端口号。如果不含则返回空串
protocol “http:” 返回页面使用的协议。通常是http:或https:
search “?q=javascript” 返回URL的查询字符串。以问号开头
