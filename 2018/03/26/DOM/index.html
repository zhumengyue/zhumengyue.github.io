DOM1级

Node 类型

节点关系

简单来说，就是父与子的关系。

每个节点都有一个 childNodes 属性，其中保存着一个 NodeList 对象。NodeList 是一个类数组对象，可通过方括号语法访问 NodeList 的值，NodeList 对象也有 length 属性。但它不是 Array 的实例。

访问节点

一般来说，可以通过方括号，也可以使用 item() 方法；

操作节点

appendchild()

插入节点，用于向 childNodes 列表的末尾添加一个节点，并返回新增的节点。

如果想把节点放在 childNodes 列表中某个特定的位置上，可以使用 insertBefore() 方法，传入两个参数：要插入的节点和作为参照的节点。

appendChild() 和 insertBefore() 方法只插入节点，不会移除节点。

replaceChild()

接收两个参数：要插入的节点和要替换的节点。

要替换的节点将由此方法返回，并从文档树中移除。

如果只需要移除节点，可以通过 removeChild() 来实现，此方法接收一个参数：即要删除的节点。

Document 类型

在浏览器中，document 对象表示整个HTML页面。Document 节点有以下属性：

  • nodeType 的值为9
  • nodeName 的值为 “#document”
  • nodeValue 的值为 null
  • parentNode 的值为 null
  • ownerDocument 的值为 null

特殊集合

document 对象有一些特殊集合，包括：

  • document.anchors ：包含所有带 name 特性的 <a> 元素
  • document.forms ：包含所有 <form> 元素，与 document.getElementsByTagName("form") 得到的结果相同
  • document.images ：包含所有 <img> 元素，与 document.getElementsByTagName("img") 得到的结果相同
  • document.links ：包含所有带 href 特性的 <a> 元素

Element 类型

  • nodeType 的值为 1
  • nodeName 的值为元素标签名

取得 / 设置 / 删除特性

创建元素

使用 document.createElement() 可以创建新元素。这个方法只接收一个参数，即要创建元素的标签名（在 HTML 文档中不区分大小写）

之后可通过 appendChild() 、insertBefore() 、replaceChild() 方法插入元素。
