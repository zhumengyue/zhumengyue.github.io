JavaScript之基本概念

技术
  • JavaScript
  • 自用笔记

　　自用笔记。

语法

区分大小写

　　ECMAScript 中的一切——变量、函数名、操作符——都区分大小写。

标识符

  1. 第一个字符必须是字母、下划线（_）、或美元符号（$）
  2. 其他字符可以是字母、下划线、美元符号、数字
  3. 采用（推荐）驼峰大小写格式，即第一个字母小写，剩下每个有意义的单词的首字母大写，比如 firstSecond 、doSomethingImportant 、myCar

注释

数据类型

　　ECMAScript 中有5种简单数据类型：Undefined、Null、Boolean、Number、String，和一种复杂数据类型—— Object。

Undefined 类型

　　Undefined 类型只有一个值，即 undefined 。在使用 var 声明变量但未对其初始化时，这个变量就是undefined。

　　变量未被初始化或未被声明，typedef 操作符都会返回 undefined，所以推荐做法是显式地初始化变量，这样当 typedef 操作符返回 undefined 时，我们可以明确知道这个变量是未声明的。

Null 类型

Boolean 类型

　　该类型有两个字面值：true 和 false。

数据类型 转换为true的值 转换为false的值
Boolean true false
String 任何非空字符串 “” （空字符串）
Number 任何非零数字值（包括无穷大） 0 和 NaN
Object 任何对象 null
Undefined undefined

Number 类型

数值范围

　　ECMAScript 所能表示的最大数和最小数保存在 Number.MAX_VALUE 和 Number.MIN_VALUE 中。如果得到一个超过此范围的数，将会自动转换成特殊的 Infinity 值。Infinity 值无法参与计算。可以通过 isFinite() 判断某个数是否在范围内。

NaN

  1. Not a Number，用于表示一个本来要返回数值的操作数未返回数值的情况（如任何数除以 0 都会都会返回NaN），这样就不会抛出错误。

  2. 任何设计 NaN 的操作都会返回 NaN。

数值转换

　　有三个函数可以将非数值转换为数值：Number() 、 parseInt() 、 parseFloat() 。

Number() 函数转换规则：

  • 如果是 Boolean 值，true 和 false 分别被转换成 1 和 0
  • 如果是数字值，只是简单地传入和返回
  • 如果是 null 值，返回 0
  • 如果是 undefined ，返回 NaN
  • 如果是字符串，忽略前导零，字符串为空时，返回 0 ；包含不可转换的字符，返回 NaN

parseInt() 函数转换规则：

  • 忽略字符串前面的空格，直到找到第一个非空字符
  • 如果第一个字符不是数字字符或负号，返回 NaN
  • 如果第一个字符是数字字符，则继续解析下一个字符，直到遇到非数字字符或解析完所有字符
  • 对于浮点数，去尾
  • 可以进行不同进制转换
  • 对于字符串，不忽略前导 0

parseFloat() 函数转换原则：

  • 只解析10进制值，所以没有指定基数的用法
  • 如果字符串包含一个可解析为整数的数（无小数点，或者小数点后全是 0 ），则会返回整数
  • 第一个小数点有效，后面的小数点无效

String 类型

  • 双引号单引号都可以，但前后要匹配
  • toString() 方法或 String() 方法将一个值转换为字符串
  • null 值和 undefined 值没有 toString() 方法
  • String() 方法原则：
    • 如果值有 toString() 方法，则调用该方法（没有参数）
    • 如果值是 null ，返回 null
    • 如果值是 undefined ， 返回 undefined

操作符

按位非

按位非（ ~ ）的结果：操作数的负值减1

逻辑与

在有一个操作数不是布尔值的情况下，逻辑与操作遵循下列规则：

  • 如果第一个操作数为对象，返回第二个操作数
  • 如果第二个操作数为对象，只有在第一个操作数的求值结果为 true 的情况下才返回该对象
  • 如果两个操作数都是对象，则返回第二个操作数
  • 如果有一个操作数为 null / NaN / undefined，则返回 null / NaN / undefined

1. 如果第一个操作数能决定结果，将不对第二个操作数求值

2. 不能在逻辑与操作中使用未定义的值

逻辑或

与逻辑与类似。

相等 && 全等

相等操作符（==）会先转换操作数，再比较相等性。全等操作符（===）直接比较。

条件操作符

variable = bool_expression ? true_value : false_value

逗号操作符

用于赋值时，返回表达式中最后一项。

语句

for-in 语句

精准的迭代语句，可以用于枚举对象属性。

ECMAScript对象属性没有顺序，所以通过 for-in 输出属性名的顺序不可预测。

Safari 3 以前版本的 for-in 语句存在bug，会导致某些属性返回两次。

label、continue、break 语句

使用 label 语句可以在代码中添加标签，以便将来配合 continue 和 break 语句使用。

大量使用 with 语句会导致性能下降，在开发大型应用程序时，不建议使用 with 语句。

switch 语句

　　switch 语句在比较值时使用全等操作符（===），因此不会发生类型转换。

函数

  1. 函数在定义时不必指定是否返回值。任何函数在任何时候都可以通过 return 语句来返回值。
  2. 位于 return 语句后的任何代码都不会被执行。
  3. return 语句可以不带任何返回值，此时函数将返回 undefined

参数

　　ECMAScript 函数不介意传递进来的参数个数和类型。即，即便定义的函数只接收两个参数，在调用时未必一定要传递两个参数，可以传递一个、三个甚至不传递参数。

没有重载

　　区别于一些语言可以通过同名不同参的函数来实现重载，ECMAScript 函数无法重载，如果定义两个同名函数，则使用后定义的函数。
